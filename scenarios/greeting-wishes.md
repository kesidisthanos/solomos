# Σενάριο: greeting-wishes

## Πότε χρησιμοποιείται
Ευχές για γενέθλια, ονομαστική εορτή, γάμο, βάφτιση, αρραβώνα, αποφοίτηση, νέα χρονιά, Πάσχα, Χριστούγεννα, ή συλλυπητήρια.

## Σκοπός & κοινό
- **Σκοπός**: να εκφράσει μια αληθινή ανθρώπινη στιγμή γραπτά, με το σωστό βάρος για την περίσταση.
- **Κοινό**: στενός (φίλος, οικογένεια, σύντροφος) ή μακρινός (γνωστός, συνάδελφος). Το register αλλάζει ανάλογα.
- **Προεπιλογή ύφους**: καθημερινό αν στενός, ουδέτερο αν απλός γνωστός ή επαγγελματικός.

## Δομή
Ελεύθερη. Τυπικά:
1. **Άνοιγμα**: απευθύνεσαι στο πρόσωπο, αναφέρεις την περίσταση αν χρειάζεται.
2. **Κορμός**: η ευχή ή η σκέψη. Όσο πιο συγκεκριμένη τόσο πιο αληθινή.
3. **Κλείσιμο**: σύντομο. Για στενό φίλο: αβίαστο. Για γνωστό: τυπικό.

Για συλλυπητήρια η δομή είναι πιο ανοιχτή: αναγνωρίζεις την απώλεια, εκφράζεις αλληλεγγύη, δεν εξηγείς και δεν παρηγορείς μηχανικά.

## Μήκος
1-5 προτάσεις τυπικά. Συλλυπητήρια λίγο μεγαλύτερα αν είναι στενός. Κάρτα: 3-8 γραμμές.

## Πρωτόκολλο πριν το γράψιμο
1. **Ποιος λαμβάνει;** Στενός ή απλός γνωστός; Αυτό καθορίζει όλα τα υπόλοιπα.
2. **Ποια η περίσταση ακριβώς;** Γενέθλια, ονομαστική, γάμος, βάφτιση, αποφοίτηση, νέος χρόνος, Πάσχα, συλλυπητήρια;
3. **Κανάλι;** Κάρτα (γραπτό, έτοιμο να εκτυπωθεί), WhatsApp, social, email;
4. **Επιθυμείς προσωπική νότα ή τυπικές ευχές;** Αν είναι στενός χωρίς καμία προσωπική νότα, ακούγεται κρύο.
5. **Για συλλυπητήρια**: Πόσο στενή η σχέση; Έχεις κάτι να πεις για τον αποθανόντα; Αν δεν τον γνώριζες, μην επιχειρήσεις εκτίμηση του χαρακτήρα του.

## Anti-patterns ειδικά για το σενάριο
- **Hallmark-translated γλυκερά**: «Σου εύχομαι όλα όσα η καρδιά σου επιθυμεί», «Να είσαι πάντα χαρούμενος/η όπως σήμερα»: calques αμερικάνικων καρτών.
- **Τυπικό «Χρόνια πολλά και του χρόνου»** σε στενό χωρίς τίποτα άλλο: αρκεί για γνωστό στο WhatsApp, αλλά για φίλο χρόνων χτυπάει ως αδιαφορία.
- **Για συλλυπητήρια**: «Είμαι σίγουρος/η ότι βρίσκεται σε καλύτερο μέρος»: ανήλικη παρηγοριά που μπορεί να προσβάλει.
- **«Τρομερή απώλεια»**: η σωστή ελληνική λέξη είναι «βαριά απώλεια».
- **Υπερβολή που δεν ταιριάζει**: «Είσαι ο καλύτερος άνθρωπος που γνώρισα ποτέ»: αν δεν είναι αλήθεια, χτυπάει ψεύτικα.
- **Em-dashes**.
- **Ψεύτικα συναισθήματα**: ευχές που γράφονται σε 10 δευτερόλεπτα και το δείχνουν. Αν δεν έχεις χρόνο για κάτι προσωπικό, γράψε λίγα και αληθινά αντί για πολλά και κοινότοπα.
- **Γενικευμένος επαινος σε συλλυπητήρια**: «Ήταν ένας εξαιρετικός άνθρωπος» αν δεν τον γνώριζες: πες αντί αυτού «Καλό δρόμο του.» ή «Να τον θυμάσαι με αγάπη.»

## Παραδείγματα

**Γενέθλια στενού φίλου:**
```
Χρόνια πολλά. Χαίρομαι που μεγαλώνεις παρέα μου.
```

**Ονομαστική γνωστού:**
```
Χρόνια πολλά, Νίκο. Να είσαι καλά.
```

**Συλλυπητήρια:**
```
Μαθαίνω για την απώλεια του πατέρα σου. Βαριά στιγμή. Είμαι εδώ αν χρειαστείς κάτι.
```

## Μορφή εξόδου

Plain text. Μερικές γραμμές, χωρίς υπερδομή. Έτοιμο για WhatsApp, κάρτα ή email ανάλογα με το κανάλι.
